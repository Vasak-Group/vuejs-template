Contiene Router y SASS pre-seteados.</p>

## Instalacion

Instalacion de dependencias del proyecto

### Node (yarn)

```bash
yarn install
```

### Bun

```bash
bun install
```

## Ejecucion

Distintos metodos para ejecutar la aplicacion ya sea que estes usando NodeJS o Bun

### NodeJS (yarn)

```bash
# watch mode
yarn run dev

# production mode
yarn run preview
```

### Bun

```bash
# watch mode
bun run dev:bun

# production mode
bun run preview:bun
```

## Build

Distintos metodos para compilar la aplicacion ya sea que estes usando NodeJS o Bun

### NodeJS (yarn)

```bash
# build for production
yarn run build
```

### Bun

```bash
# build for production
bun run build:bun
```
